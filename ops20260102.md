# 🚦 보행 환경 특화: 로봇 배송 HRI 접근성 체크리스트
**Project: Robot Delivery Service in Pedestrian Environments**
**Target Device:** DJI RoboMaster S1 (Python Control)

본 문서는 일반적인 화면(Screen) UI를 넘어, 보행 환경에서 로봇의 **불빛(LED), 소리(Audio), 움직임(Movement), 시선(Gimbal)** 등 물리적 행동을 포함한 포괄적 UI(User Interface) 가이드라인을 정의합니다.

---

## 👵 1. 고령자 (Seniors)
> **핵심 키워드:** 공포감 해소, 명확한 인지, 물리적 안전

고령자는 시청각 능력이 저하되어 있을 수 있으며, 로봇과 같은 낯선 기계에 대해 두려움(Technophobia)을 느낄 확률이 높습니다. **안심할 수 있는 피드백**이 최우선입니다.

| 구분 | 접근성 체크리스트 (UI Issue) | RoboMaster S1 구현 방안 (Python) |
| :--- | :--- | :--- |
| **청각 UI** | 고주파 기계음(Beep) 대신 **사람 목소리나 낮은 톤의 경적**을 사용하는가? | `media_ctrl.play_sound()` : 녹음된 음성 안내("비켜주세요", "지나갑니다") 재생 |
| **시각 UI** | 로봇의 **현재 상태(정지/이동)**가 멀리서도 빛으로 구별되는가? | `led_ctrl` : 정지 시 빨간색 점멸(경고), 이동 시 초록색 상시 점등(안전) 구분 |
| **운동 UI** | 보행자 발견 시 급정거하지 않고 **부드럽게 감속**하는가? | `chassis_ctrl` : 감속 시간을 0.5초 이상 두어 부드럽게 멈춤 (Soft Stop) |
| **거리감** | 보행자와 충분한 **안전거리(Proxemics, 1.5m 이상)**를 유지하는가? | 비전 마커/사람 인식 거리가 일정 수준 이내면 즉시 우회 경로 생성 |

## 🧒 2. 청소년/어린이 (Adolescents/Kids)
> **핵심 키워드:** 예측 불가능성 대응, 직관적 신호, 안전 사고 방지

스마트폰을 보며 걷거나(Smombie), 로봇에 대한 호기심으로 진로를 방해하는 돌발 행동에 대응해야 합니다.

| 구분 | 접근성 체크리스트 (UI Issue) | RoboMaster S1 구현 방안 (Python) |
| :--- | :--- | :--- |
| **시각 UI** | **방향 지시등(Turn Signal)**으로 진행 방향을 미리 알리는가? | `led_ctrl` : 좌회전 시 왼쪽 LED만 노란색으로 깜빡임 (차량 UI 차용) |
| **반응 UI** | 진로 방해 시 공격적이지 않게 **우회 요청**을 하는가? | `gimbal_ctrl` : 짐벌을 좌우로 흔들며 곤란하다는 제스처 표현 (Social Gesture) |
| **안전 UI** | 로봇에 충격을 가하거나 건드렸을 때 **즉시 작동을 멈추는가**? | `armor_ctrl` : 충격 센서 감지 시 모든 모터 전원 차단 (Emergency Stop) |
| **주목성** | 스마트폰을 보는 보행자를 위해 **바닥 조명** 등을 활용하는가? | (추가 장착) 빔 프로젝터나 레이저로 바닥에 진행 화살표 표시 |

## 👩‍💼 3. 성인 (Adults)
> **핵심 키워드:** 사회적 매너(Social Norms), 정보의 명확성

로봇이 보행자로서의 규칙(우측통행 등)을 지키고, 자신의 역할을 명확히 하기를 기대합니다.

| 구분 | 접근성 체크리스트 (UI Issue) | RoboMaster S1 구현 방안 (Python) |
| :--- | :--- | :--- |
| **사회적 UI** | 좁은 길 교행 시, **먼저 멈추고 길을 양보**하는가? | `chassis_ctrl` : 정면 충돌 코스 감지 시 우측으로 틀어 정지 및 대기 |
| **시선 UI** | 로봇이 **보행자를 인지하고 있음**을 시선으로 표현하는가? | `vision_ctrl` : 사람 얼굴/마커 좌표를 따라 짐벌이 시선을 맞춤 (Eye Contact) |
| **정보 UI** | 현재 **'배송 중'인지 '순찰/대기 중'인지** 구분되는가? | `led_ctrl` : 배송 중(파랑), 복귀 중(노랑), 오류(빨강) 등 색상 코드 정의 |

## 💻 4. Python 구현 예시 (Code Snippet)

위의 체크리스트를 바탕으로 작성된 **[고령자 친화적 정지 및 인사]** 코드 예시입니다.

# RoboMaster S1 Python Code: Senior-Friendly Safe Stop Behavior

def safe_stop_behavior():
    """
    고령자나 보행자를 만났을 때 위협감을 주지 않고 
    안전하게 멈추며 의사를 표현하는 함수
    """
    
    # 1. [운동 UI] 확 서지 않고 부드럽게 감속 (Jerk 최소화)
    # 현재 속도에서 0까지 0.5초간 서서히 줄이는 로직 필요
    chassis_ctrl.set_wheel_speed(30, 30, 30, 30) # 1차 감속
    time.sleep(0.3)
    chassis_ctrl.stop() # 완전 정지
    
    # 2. [시각 UI] "멈췄음"을 알리는 빨간불 천천히 깜빡임 (심박수 리듬)
    # effect_breath: 숨쉬는 듯한 부드러운 점멸 효과
    led_ctrl.set_top_led(rm_define.armor_top_all, 255, 50, 50, rm_define.effect_breath)
    
    # 3. [시선 UI] 고개를 살짝 숙여서 위협적이지 않음을 표현 (인사/양보)
    gimbal_ctrl.set_rotate_speed(0, -20) # 아래로 Pitch 이동
    time.sleep(0.5)
    gimbal_ctrl.set_rotate_speed(0, 0)
    
    # 4. [청각 UI] 기계음 대신 부드러운 알림음 또는 녹음된 음성
    media_ctrl.play_sound(rm_define.media_sound_recognize_success)

# 실행 예시: 사람 감지 시 호출
